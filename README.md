## 🛠️ Troubleshooting

### The app does not open

Try these steps:

- Right-click the app and choose Run as administrator
- Check that the file finished downloading
- Move the file to your Desktop or Downloads folder
- Restart your PC and try again

### Windows blocks the file

Try this:

- Right-click the file
- Open Properties
- Look for an Unblock option
- Apply the change if you see it
- Run the file again

### The app opens but looks empty

Try these steps:

- Wait a few seconds for content to load
- Check your internet connection
- Close the app and open it again
- Download the latest release from the release page

### The app closes right away

Try these steps:

- Restart the app
- Restart Windows
- Make sure you downloaded the full file
- Try the newest release from the download page
